# Flappy Bird NEAT

Training the computer to play flappy bird using NeuroEvolution of Augmentating Topologies (NEAT).

Requirements
* pygame 1.9.6
* neat-python 0.92
* python 3.7.6

Use
* ```main.py``` for playing the game.
* ```neat_train.py``` for training the game.
